This demo showcases a custom <my-input> web component that fully integrates with forms just like native input elements.

Key Features Demonstrated

1. Form Participation

The <my-input> component submits values with the form and participates in form validation.

2. Programmatic API

The component exposes the same API as a native input element.

3. Form Reset Support

The component responds to form resets just like a native input.

4. Custom Styling

The component can be styled with CSS parts.
